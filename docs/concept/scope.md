# Escopo

O **escopo** é um conjunto de informações que são usadas para manter um contexto sobre alguma preferencia, estado ou informação que pode  ou não ser alterada durante a execução de um Snipet.

Ele possui algumas configurações, como:
- **nome**: o nome do escopo
- **descrição**: a descrição do escopo
- **volatilidade**: se o escopo é volátil ou não (pode ser alterado durante a execução do Snipet)
